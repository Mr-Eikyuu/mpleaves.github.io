# javaDoc 生成文档

  • javadoc 命令是用来生成自己 API 文档的

# 参数信息

  • @author 作者名
  • @version 版本号
  • @since 指明需要最早使用的 jdk 版本
  • @param 参数名
  • @return 返回值情况
  • @throws 异常抛出情况
