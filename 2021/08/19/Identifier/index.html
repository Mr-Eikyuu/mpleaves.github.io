# 标识符

  1. 所有的标识符都应该以字母（A-Z 或者 a-z），美元符号（$）、或者下划线（_）开始
  2. 首字符之后可以是字母（A-Z 或者 a-z），美元符号（$）、下划线（_）或数字的任何字符组成
  3. 不能使用关键字做为变量名或方法名
  4. 标识符大小写敏感
  5. 合法标识符举例：age、$salary、_value
  6. 非法标识符举例：123abc、-salary、#abc
