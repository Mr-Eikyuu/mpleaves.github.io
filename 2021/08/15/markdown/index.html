# Markdown 学习

一级标题：一个 #后写内容

# 二级标题

二级标题：两个 #后写内容

# 三级标题

三级标题：三个 #后写内容

# 字体

粗体：Hello,World 两边加两个星号

斜体：Hello,World 两边加一个星号

既是粗体也是斜体：Hello,World 两边加三个星号

删除线效果：Hello,World 两边加两个波浪线

# 引用

Hello World 前面加右箭头

# 分割线

分割线：三个减号或者三个星号回车

# 图片

图片：前面一个感叹号＋中括号＋小括号中间写本地路径或者网页路径

# 超链接

点击跳转到 mpleaves 的个人博客

超链接：中括号里写内容，紧接着写一个小括号里面写链接

# 列表

  1. A
  2. B
  3. C
  4. D

有序列表：前面写数字，后面加点和空格

  • A
  • B
  • C
  • D

无序列表：前面写减号加空格

# 表格

名字 性别 生日
mpleaves 男 2002.6.11
表格：元素 元素 … 回车继续写– – – … 回车继续写内容

# 代码

代码：三个 ` 后写什么编程语言，比如 Java 然后写内容
